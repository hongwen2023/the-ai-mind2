# The AI Mind

一套长期、开源、第一性原理的 AI 课程 —— 从 AI Learner 到 Engineer、Researcher、Builder。

> 中心对象：**智能 = 在不确定性与资源约束下从经验中获得达成目标的能力；学习 = 用数据做归纳推断。**

- 课程总架构、Prelude、Chapter 1、自我评估
